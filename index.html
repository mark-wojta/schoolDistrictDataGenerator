Purpose of Data Generator
This website allows users to select on a map, or query in a search bar, a specified school district in Wisconsin. Depending on which school district is queried or selected, this website will output data from the 2018-2019 school year such as general school district, enrollment, attendance, and disciplinary information. Output school district data could be used for relocation, community strengthening, organizational, or general data needs. Users are able to enter addresses to examine surrounding public libraries, public schools, and school districts. The map allows the users to toggle the public libraries layer, public schools layer, and school districts layer, off and on. Users can also search public libraries and public schools by name.

Most recent yearly school district data will be updated on this website when available.
